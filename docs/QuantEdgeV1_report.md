# QuantEdge V1

## Overview

QuantEdge is a Python-based quantitative Smart Money Concepts (SMC) research engine that automates market structure analysis, liquidity detection, signal generation, and strategy backtesting.

---

## Pipeline

1. Swing Detector V3
2. Market Structure V4
3. BOS Detection
4. CHOCH Detection
5. Liquidity Detection
6. Liquidity Sweep Detection
7. Validated Sweeps
8. Order Block Detection
9. Fair Value Gap Detection
10. Signal Generation
11. Backtesting Engine
12. Equity Curve Generation
13. Drawdown Analysis
14. Performance Analysis

---

## Markets Tested

- EURUSD M15 (2 Years)
- GBPUSD M15 (2 Years)
- XAUUSD M15 (2 Years)
